# Controle Financeiro em Python

Projeto de controle financeiro desenvolvido em Python.

## Funcionalidades

- Cadastro de receitas e despesas
- Dashboard financeiro
- Gráficos com Matplotlib
- Análise de dados com Pandas
- Exportação Excel
- Persistência JSON

## Tecnologias

- Python
- Pandas
- Matplotlib
- OpenPyXL
